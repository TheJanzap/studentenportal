Dieses Portal ist für alle Studenten der Hochschule Rapperswil gedacht. Hier könnt ihr Informationen für euer Studium bekommen, News zu Events und Ähnlichem erfahren und euer Wissen mit anderen Studenten teilen.

News

17.02.13: Den Dokumenten können seit längerem auch URLs (zB auf Link auf das Github-Projekt mit LaTeX-Quellen) zugewiesen werden. Neu kann man optional auch die Lizenz (Creative Commons oder Public Domain) festlegen.
Desweiteren wurde Uservoice entfernt, es hat sich nicht bewährt. Stattdessen Bugs und Featureideen weiterhin auf Github erfassen.

28.01.13: Für die Dokumente gibt es nun zwei neue Kategorien: Software (zB für Taschenrechner-Programme) und Lernhilfen (zB für Lern-Karteikarten oder Repetitionsfragen).

26.07.12: Neu gibt es für das Studentenportal eine API (bisher readonly). Authentifizierung läuft momentan via manuell generierter API Keys. Wer einen solchen Key möchte, bitte ein Mail an {{ 'dbargen@hsr.ch'|urlize }} oder ein Tweet an @studportal_hsr senden.

16.06.12: Es gibt nun Flattr Buttons bei den Zusammenfassungen. Um den Button für deine Dokumente zu aktivieren, gib deinen Flattr-Benutzernamen in deinem Profil an.

Mitmachen

Dieses neue Studentenportal wurde entwickelt, weil die vorherige Seite veraltet und voller Sicherheitslücken war. Die Entwicklung ist noch im Gange, wenn du Python oder HTML/CSS kennst oder es gerne lernen möchtest, kannst du uns gerne auf Github forken.

Um mitzumachen musst du aber nicht programmieren können. Auch zB designtechnische Verbesserungen wären cool und viele Zusammenfassungen und Zitate machen die Seite interessant.

Bugs und Ideen

Wenn Fehler auftreten oder wenn du eine coole Idee fürs Studentenportal hast, dann erfasse bitte ein Ticket auf Github oder schreibe eine E-Mail an {{ 'dbargen@hsr.ch'|urlize }}. Die momentan geplanten Features findest du hier.
